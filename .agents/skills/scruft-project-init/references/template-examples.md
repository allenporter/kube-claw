# Template Examples

## Allen Porter's Cookiecutter Python
- **URL**: `https://github.com/allenporter/cookiecutter-python`
- **Features**:
  - `uv` for dependency management
  - `ruff` for linting and formatting
  - `mypy` for type checking
  - GitHub Actions for CI/CD
  - `pre-commit` hooks
  - Standardized `./script/` directory (setup, test, lint)

## Initialization Parameters
When prompted by `scruft create`, common parameters include:
- `project_name`: Human-readable name
- `project_slug`: Folder/package name (e.g., `kube_claw`)
- `author_name`: e.g., `Allen Porter`
- `github_username`: e.g., `allenporter`
